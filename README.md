# Yorkshire & Humber Crime Analysis

## Overview
This project analyzes crime data in the Yorkshire & Humber region using interactive Power BI dashboards.  
The goal is to uncover crime patterns, trends, and outcomes to support better understanding of regional safety and policing effectiveness.

---

## Tools & Technologies
- SQL Server  
- T-SQL  
- Power BI  
- Data Cleaning & Transformation  
- Data Visualization  

---

## Key Features
- Overall Crime Analysis (1.3M+ total records analyzed)  
- Crime Type Distribution (Anti-social behavior, burglary, bicycle theft, etc.)  
- Monthly & Yearly Trends in crime occurrences  
- Geospatial Crime Mapping using latitude & longitude  
- Crime Outcome Analysis (arrests, unresolved cases, prosecutions)  
- Stop & Search Insights (demographics, success rates, search types)  
- Police Force Comparison across regions (West Yorkshire, South Yorkshire, etc.)  
- LSOA-Level Analysis for localized crime insights  

---

## Dataset
UK Police Street Crime & Stop and Search Data  
https://data.police.uk/data/

---

## Key Insights

### Crime Overview
- Over **1.34 million crimes** recorded in the dataset  
- **Violence & sexual offences (~39.5%)** are the most common crime category  
- Crime levels remain relatively consistent throughout the year, with slight peaks mid-year  

### Crime Outcomes
- ~38–44% of cases result in **“Unable to prosecute”**, indicating investigation challenges  
- A large portion of cases also fall under **“Under investigation” or unresolved categories**  
- Arrest rates remain comparatively low  

### Stop & Search Analysis
- **63K+ stop & search incidents** recorded  
- **Person search (~74.7%)** is the most common search type  
- **Successful arrest rate is only ~16.2%**, suggesting low conversion efficiency  
- Majority of searches involve individuals aged **18–34**  

### Demographic Insights
- Highest proportion of stops involve individuals identified as **White ethnicity (~60% officer-defined, ~42% self-defined)**  
- Smaller proportions across other ethnic groups  

### Regional Insights
- **West Yorkshire reports the highest number of crimes**  
- Crime distribution varies significantly across **LSOA regions**, with certain hotspots dominating  

### Crime Patterns
- Anti-social behavior and theft-related crimes are consistently high  
- Monthly trends show **stable but persistent crime levels**, not sharp fluctuations  
- Some seasonal variation is visible but not extreme
